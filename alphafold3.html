핵심 변화 요약 (Key Changes)

✕
MSA 트랙 축소: Evoformer의 48블록 MSA 처리 → 단 4블록 전처리로 대폭 축소. 진화 정보는 Pair 표현으로 압축.
↻
Evoformer → Pairformer: MSA+Pair 이중 트랙 → Single+Pair 이중 트랙. MSA 표현(Nseq×Nres) 대신 Single 표현(n×384)으로 교체.
✓
확산 모델 도입: IPA 구조 모듈(결정론적, 프레임 기반) → 확산 디노이징(확률적, 원자 좌표 직접 예측). DALL·E처럼 노이즈에서 구조를 생성.
✓
범용 예측: 단백질만 → 단백질, DNA, RNA, 소분자, 이온, 공유결합 변형까지 모든 생체분자 복합체 예측 가능.
✕
SE(3) 등변성 제거: AF2의 IPA는 SE(3) 등변성을 엄격히 유지 → AF3의 확산 모듈은 등변성이 필요 없음! 아키텍처 단순화.

Evoformer → Pairformer 진화

AF2: Evoformer (×48 블록)

핵심 구조: 두 개의 큰 병렬 트랙이 정보를 양방향으로 교환합니다.
• MSA 표현: Nseq × Nres × 256 — 수천 개의 상동 서열 정보
• Pair 표현: Nres × Nres × 128 — 잔기 쌍 관계
• 48블록에 걸쳐 MSA↔Pair 상호 정보 교환 (Row/Column Attention, Outer Product Mean, Triangle Updates)

AF3: MSA Module(×4) + Pairformer(×48)

핵심 변화: MSA 정보를 전처리 단계에서 Pair 표현으로 압축한 후, 메인 트렁크에서는 MSA를 사용하지 않습니다.
• MSA Module (×4): MSA 정보를 빠르게 Pair 표현에 주입
• Single 표현: n × cs(=384) — MSA 표현을 대체
• Pair 표현: n × n × cz(=128) — AF2와 동일한 Triangle Updates 유지
• "Single attention with pair bias" — 새로운 어텐션 메커니즘

구조 모듈 → 확산 모듈 (Diffusion Module)

AF2: 구조 모듈 (Structure Module)

특징:
• IPA (Invariant Point Attention) 사용
• SE(3) 등변성 유지
• 백본 프레임 (회전+이동) 기반
• 8회 반복 정제 (deterministic)
• 같은 입력 → 항상 같은 출력
• 토션 각도로 곁사슬 별도 처리
• Violation loss 필요 (화학적 제약)

AF3: 확산 모듈 (Diffusion Module)

디노이징 스텝: 0 / 20 | 노이즈 레벨: 100%
특징:
• 디노이징 확산 (Denoising Diffusion) 접근
• SE(3) 등변성 불필요! (단순화)
• 원자 좌표 직접 예측 (프레임 없음)
• 20 디노이징 스텝 (stochastic)
• 다른 시드 → 다른 유효 구조 (생성 모델)
• 모든 원자 직접 처리 (단백질+DNA+리간드)
• Violation loss 불필요

예측 범위 확장 (Expanded Prediction Scope)

AlphaFold2: 단백질만

단백질 단량체 (Monomer) 단백질 다량체 (Multimer) DNA RNA 소분자 (Ligand) 이온 (Ion)

AlphaFold3: 모든 생체분자

단백질 (Protein) DNA RNA 소분자/리간드 (Ligand) 이온 (Ion) 변형 잔기 (Modified residues) 공유결합 변형 (Covalent mods)
왜 중요한가? 생물학적으로 의미 있는 대부분의 과정은 단백질이 다른 분자(DNA, 약물, 금속 이온 등)와 상호작용할 때 발생합니다. AF2는 단백질 구조만 예측할 수 있어 이런 복합체를 이해하는 데 한계가 있었습니다. AF3는 하나의 모델로 모든 생체분자 복합체를 예측할 수 있어, 약물 설계(단백질-리간드), 유전자 조절(단백질-DNA), RNA 생물학 등 광범위한 응용이 가능합니다.

신뢰도 & 환각 방지 (Confidence & Anti-Hallucination)

신뢰도 지표 (Confidence Metrics)

pLDDT
잔기별 구조 정확도 (per-residue accuracy). AF2와 동일.
PAE
정렬 오류 예측 (Predicted Aligned Error). 도메인 간 상대 위치 신뢰도.
PDE
신규! 쌍별 거리 오류 (Pairwise Distance Error). AF3에서 새로 추가.
Mini-Rollout 전략: 확산 모델의 훈련 중에는 최종 구조를 직접 회귀할 수 없습니다 (노이즈가 있는 중간 상태에서 훈련하므로). 따라서 훈련 중 "mini-rollout"을 수행하여 — 즉, 부분적으로 디노이징을 실행하여 — 신뢰도 모듈이 최종 구조의 품질을 예측하도록 학습합니다.

교차 증류로 환각 방지 (Cross-Distillation)

문제: 확산 모델은 생성 모델이므로, 무질서한 영역(disordered regions)에서도 그럴듯한 구조를 "환각(hallucinate)"할 위험이 있습니다.
해결책 — 교차 증류 (Cross-Distillation): AF-Multimer v2.3에서 생성한 예측 구조를 추가 훈련 데이터로 사용합니다. AF2가 무질서한 영역에서 낮은 pLDDT를 출력하면, AF3도 해당 영역에서 "모른다"고 표현하는 법을 배웁니다. 이를 통해 환각을 크게 줄입니다.

핵심 통찰: AF2의 "약점"(무질서 영역 예측 실패)이 AF3의 "교사 신호"로 재활용됩니다.
