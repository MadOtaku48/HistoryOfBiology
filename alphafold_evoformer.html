Evoformer — 진화 정보에서 구조 정보를 추출하다

단백질의 아미노산 서열은 진화 과정에서 공진화(coevolution) 패턴을 남깁니다. 구조적으로 가까이 접촉하는 잔기들은 함께 변이하는 경향이 있기 때문입니다. Evoformer는 수천 개의 상동 서열(homologous sequences)로 구성된 다중 서열 정렬(MSA)로부터 이러한 공진화 패턴을 읽어내어, "어떤 잔기 쌍이 3D 공간에서 가까이 있는가"라는 구조적 정보를 추출합니다. 이 과정은 MSA 표현과 쌍(Pair) 표현 사이의 양방향 정보 교환을 48번 반복하며, 반복할수록 더 정교한 구조 정보가 축적됩니다.

MSA 표현 (Multiple Sequence Alignment Representation)

다중 서열 정렬(MSA)은 진화적으로 관련된 단백질 서열들을 정렬한 것입니다. 행(row)은 각 서열, 열(column)은 잔기(residue) 위치를 나타냅니다. Evoformer는 행 방향 어텐션(같은 위치에서 서열 간 정보 교환)과 열 방향 어텐션(같은 서열 내 위치 간 패턴 학습)을 번갈아 수행합니다. 텐서 형태: (Nseq × Nres × 256)

쌍 표현 (Pair Representation)

쌍 표현은 모든 잔기 쌍 (i, j) 사이의 관계를 인코딩하는 2D 행렬입니다. 텐서 형태: (Nres × Nres × 128). 삼각 업데이트(Triangular Update): 쌍 (i,j)의 정보를 업데이트할 때, 제3의 잔기 k를 경유하는 삼각형 경로 (i,k)→(k,j)를 통해 정보가 전달됩니다. 이는 삼각 부등식(거리 관계의 일관성)을 보장합니다. 외적 평균(Outer Product Mean)을 통해 MSA의 정보가 쌍 표현으로 전달됩니다.

양방향 정보 교환 (Bidirectional Exchange)

Evoformer 블록 내에서 MSA 표현과 쌍 표현은 양방향으로 정보를 교환합니다. 이 과정이 48개 블록에 걸쳐 반복되며, 서열 정보와 구조 정보가 점진적으로 정제됩니다.

블록 (Block): 0 / 48 반복

행 방향 어텐션 (Row-wise Attention)

각 잔기 위치(열)에서, 서로 다른 서열들이 정보를 교환합니다. 이를 통해 "이 위치에서 진화적으로 어떤 아미노산이 보존되는가?"를 학습합니다. 쌍 표현의 편향(bias)이 추가되어 구조 정보가 MSA 업데이트에 반영됩니다.

Row Attention with Pair Bias

열 방향 어텐션 (Column-wise Attention)

각 서열(행) 내에서, 잔기 위치들이 서로 정보를 교환합니다. "이 서열에서 위치 i와 위치 j는 어떤 관계인가?"를 학습하며, 이는 공진화(coevolution) 패턴을 포착하는 핵심 메커니즘입니다.

Column Attention

삼각 곱셈 업데이트 (Triangular Multiplicative Update)

쌍 (i,j)를 업데이트할 때 중간 잔기 k를 경유합니다. "Outgoing": (i,k)와 (j,k)로부터 → (i,j) 업데이트. "Incoming": (k,i)와 (k,j)로부터 → (i,j) 업데이트. 이는 그래프에서 삼각형 닫힘(triangle closure)과 유사합니다.

Triangular Multiplicative Update

삼각 어텐션 (Triangular Attention)

"Starting node": 행 i를 공유하는 모든 쌍 (i,j)와 (i,k) 간의 어텐션. "Ending node": 열 j를 공유하는 모든 쌍 (i,j)와 (k,j) 간의 어텐션. 이를 통해 쌍 표현의 일관성이 유지됩니다.

Triangular Self-Attention

왜 삼각 업데이트가 필요한가? — 거리의 기하학적 일관성

쌍 표현은 잔기 i와 j 사이의 관계(거리, 접촉 등)를 인코딩합니다. 그런데 단백질은 3D 공간에 존재하는 물리적 객체이므로, 모든 잔기 쌍의 거리는 반드시 삼각 부등식을 만족해야 합니다:

d(i, j) ≤ d(i, k) + d(k, j)
임의의 세 잔기 i, j, k에 대해 항상 성립

만약 쌍 표현을 각 (i,j)에 대해 독립적으로만 업데이트하면 어떻게 될까요? 아래 그림에서 삼각 부등식이 만족되는 경우와 위반되는 경우를 비교해 보세요.

구체적으로 두 방향의 삼각 업데이트가 번갈아 적용됩니다:

Outgoing Edges
(i,k) × (j,k) → (i,j)
k에서 나가는(outgoing) 간선들의
정보를 합산하여 (i,j) 갱신
Incoming Edges
(k,i) × (k,j) → (i,j)
k로 들어오는(incoming) 간선들의
정보를 합산하여 (i,j) 갱신

핵심 의의: 삼각 업데이트는 "단백질 구조는 유클리드 공간에 임베딩 가능해야 한다"는 물리적 제약을 신경망 아키텍처에 직접 반영한 귀납적 편향(inductive bias)입니다. 이 설계 덕분에 Evoformer는 단순한 패턴 매칭이 아닌, 물리적으로 실현 가능한 3D 구조에 대응하는 거리 행렬을 학습할 수 있습니다.

Structure Module — 추상적 정보를 실제 3D 구조로 변환하다

Evoformer가 추출한 구조 정보는 아직 추상적인 수치(텐서)일 뿐, 실제 원자의 3D 좌표가 아닙니다. Structure Module은 이 정보를 받아 각 잔기에 로컬 좌표 프레임(회전 + 이동)을 부여하고, IPA(Invariant Point Attention)를 통해 3D 공간 정보와 추상적 특징을 동시에 활용하여 프레임을 반복적으로 업데이트합니다. 처음에는 무작위 상태에서 시작하지만, 8회 반복을 거치며 단백질이 마치 실제로 접히듯 점진적으로 올바른 3D 구조로 수렴합니다. 최종적으로 백본(backbone) 좌표가 결정되면, 각 잔기의 곁사슬(side chain) 비틀림 각도(χ angle)를 예측하여 완전한 단백질 구조를 생성합니다.

IPA: 불변점 어텐션 (Invariant Point Attention)

IPA는 Structure Module의 핵심입니다. 각 잔기는 로컬 좌표 프레임(회전 + 이동)을 가지며, 어텐션 계산에 추상적 특징과 3D 위치 정보를 모두 사용합니다. 핵심 성질: 전체 구조를 회전/이동시켜도 어텐션 값이 변하지 않습니다 (SE(3) 불변성).

IPA는 왜 필요하고, 어떻게 작동하는가?

일반적인 Transformer의 어텐션은 추상적 벡터(임베딩)만으로 "이 잔기와 저 잔기가 얼마나 관련 있는가?"를 계산합니다. 하지만 단백질 구조를 예측하려면 3D 공간에서의 실제 위치도 고려해야 합니다. 예를 들어, 서열상 멀리 떨어진 두 잔기가 3D에서는 매우 가까울 수 있습니다. IPA는 이 두 가지 정보를 동시에 사용합니다.

IPA 어텐션의 3가지 구성 요소
① 일반 어텐션
Evoformer가 만든 추상적 특징 벡터 si로부터 Query, Key, Value를 계산합니다. 이것은 일반적인 Transformer와 동일합니다.
qi · kj
② 쌍 표현 편향
Evoformer의 쌍 표현 zij가 어텐션에 편향(bias)으로 추가됩니다. "이 두 잔기는 가까울 것이다"라는 정보가 반영됩니다.
+ bij (from zij)
③ 불변점 (핵심!)
각 잔기 i의 로컬 프레임 Ti를 사용하여, 추상적 점(point)을 3D 글로벌 좌표로 변환한 뒤, 두 잔기의 점 사이 유클리드 거리를 어텐션에 반영합니다.
‖ Ti(pi) − Tj(pj) ‖²
왜 "불변(Invariant)"인가?

핵심 트릭은 로컬 프레임의 사용입니다. 어텐션 점수를 계산할 때, 각 잔기는 자신의 로컬 좌표계에서 정의된 점(query point, key point)을 글로벌 좌표로 변환(Ti 적용)한 뒤, 두 점 사이의 거리를 측정합니다.

단백질 전체를 회전(R)하거나 이동(t)시키면, 모든 프레임 Ti가 동일하게 변환되므로 두 점 사이의 상대적 거리는 변하지 않습니다. 따라서 어텐션 값도 동일하게 유지됩니다.

글로벌 변환 적용
Ti → R·Ti + t
Tj → R·Tj + t
↓
‖ (R·Ti+t)(p) − (R·Tj+t)(p) ‖
= ‖ Ti(p) − Tj(p) ‖
거리 불변! → 어텐션 불변!

왜 중요한가: 단백질의 기능과 구조는 내부 원자 간의 상대적 배치에 의해 결정되며, 실험실에서 단백질을 어느 방향으로 놓든 같은 단백질입니다. IPA의 SE(3) 불변성은 이 물리적 사실을 신경망에 직접 반영합니다. 덕분에 네트워크는 임의의 좌표계에서 학습해도 물리적으로 동일한 구조에 대해 동일한 결과를 보장하며, 데이터 증강(회전 augmentation) 없이도 올바른 3D 구조를 학습할 수 있습니다.

백본 정제 (Backbone Refinement)

Structure Module은 8회 반복(iteration)을 통해 단백질 백본(backbone)을 정제합니다. 각 잔기의 프레임(frame)이 처음에는 무작위 상태에서 시작하여 점진적으로 올바른 3D 구조로 수렴합니다. 각 반복마다 IPA를 통해 업데이트된 프레임이 더 정확한 구조를 만들어냅니다.

반복 (Iteration): 0 / 8

곁사슬 예측 (Side Chain Prediction - χ Angles)

백본 구조가 결정된 후, 각 잔기의 곁사슬(side chain)이 비틀림 각도(χ angles)를 통해 예측됩니다. 백본 프레임으로부터 곁사슬이 "자라나며", 최종적으로 원자 수준의 전체 구조가 완성됩니다.

SE(3) 불변성 (Invariance)

IPA의 어텐션은 글로벌 회전과 이동에 대해 불변입니다. 단백질을 어떻게 회전시키든 동일한 어텐션 가중치가 계산됩니다. 이는 물리적으로 의미 있는 예측을 보장합니다.

Invariant Point Attention

프레임 업데이트 (Frame Update)

각 잔기의 프레임 T = (R, t)는 회전 행렬 R과 이동 벡터 t로 구성됩니다. IPA 출력을 기반으로 프레임이 업데이트되며, 8회 반복 후 최종 구조가 결정됩니다.

Backbone Update

비틀림 각도 (χ Angles)

곁사슬은 최대 4개의 비틀림 각도(χ1, χ2, χ3, χ4)로 정의됩니다. ResNet 기반 예측기가 각 잔기의 곁사슬 형태를 결정합니다.

Torsion Angle Prediction
